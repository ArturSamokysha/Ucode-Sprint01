Protocols

http/https - широко распространённый протокол передачи данных, изначально предназначенный для передачи гипертекстовых документов

DNS - это компьютерная четко распределенная система для получения информации о состоянии домена

IP - маршрутизируемый протокол сетевого уровня стека TCP/IP
